Pure Vector vs. Graph Walk
🔍 Pure Vector Search
Stops at step 3. Returns only the nearest 5 nodes. No cross-domain synthesis. Limited context.
🕸️ Graph Walk
Continues to step 4. Follows organic edges. Discovers connected knowledge across domains. Rich context assembly.
Key Design Principles
⚡
Local First
Steps 1-5 run entirely offline. Only reasoning requires external LLM. Total local latency: ~50ms.
🕸️
Organic Connectivity
Graph edges built by extraction + think cycles. No manual curation. Emergent cross-domain links.
🎯
Smart Traversal
BFS respects edge weights. Cosine similarity guides neighbor selection. Quality over quantity.
